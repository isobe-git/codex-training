# 銘柄管理アプリ（MVP）

株式投資運用向けの銘柄管理アプリです。初期版は**手入力の株価更新**で運用し、将来のAPI自動取得へ拡張しやすい構成にしています。

## 主要機能

- 保有銘柄一覧（証券会社別、平均取得単価・現在値・評価損益）
- 取引履歴（証券会社、買付/売却、数量、単価、手数料、日付）
- 目標株価・損切りラインアラート
- 3σ（ボリンジャーバンド上限/下限）表示と判定
- トレーリングストップ設定
- 月次/年次の実現損益レポート
- 銘柄マスタ管理（銘柄コード / 銘柄名 / 市場 / 業種）
- CSVインポート / エクスポート
- 単一ユーザー前提（認証なし）

## 使い方

1. `index.html` をブラウザで開く（または簡易HTTPサーバーで配信）
2. 最上部の「ダッシュボード」で評価損益・実現損益・アラートの概要を確認
3. 「保有銘柄」で証券会社別の保有状況を確認
4. 「月次 / 年次レポート（実現損益）」で実現損益の推移を確認
5. 「価格を更新」で現在値を入力
6. 「取引を追加」で証券会社を含む売買履歴を登録
7. 「アラート設定」で目標/損切り/3σ/トレーリングを登録
8. 「取引履歴」で登録内容を確認
9. 「銘柄マスタ管理」で銘柄情報を登録・更新
10. 「銘柄マスタ一覧」でマスタ登録結果を確認
11. 最下部の「CSVインポート / エクスポート」でバックアップ/復元を実行

## CSV形式

`type` 列でレコード種別を表現する単一CSVです。

- `MASTER`: 銘柄マスタ
- `TRADE`: 取引履歴
- `PRICE`: 価格履歴
- `ALERT`: アラート設定

## 将来拡張

- `PriceProvider` 抽象を使って、手入力 (`ManualPriceProvider`) から API 取得 (`ApiPriceProvider`) に差し替え可能
- 通知を画面内表示からPush/メールへ拡張可能

## 開発

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` にアクセス。
